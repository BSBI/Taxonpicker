# taxonpicker

This repo implements a bare-bones taxon picker widget

*See also https://github.com/BSBI/Taxonpicker*

## Getting started

Clone this repository and install its dependencies:

```bash
git clone https://github.com/BSBI/Taxonpicker
cd taxonpicker
npm install
npm run build
```
